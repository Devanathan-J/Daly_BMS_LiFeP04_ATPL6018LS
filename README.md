# Daly_BMS_LiFeP04_ATPL6018LS
Daly_BMS_LiFePO₄_ATPL6018LS is used for research and development to study LiFePO₄ battery management in electric vehicles. The work focuses on protection mechanisms, cell balancing, SoC/SoH analysis, thermal behavior, and safe integration of battery packs for EV applications.
